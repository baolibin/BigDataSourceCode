* AbstractList：这个类提供了{@link List}接口的框架实现，以最小化实现这个由“随机访问”数据存储（如数组）支持的接口所需的工作量。对于顺序访问数据（如链表），应优先使用{@link AbstractSequentialList}而不是此类。
